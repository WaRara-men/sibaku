# しばく！！ (SHIBAKU!!)

## アプリ概要
「しばく！！」は、日々のハプニングやストレスを連打で「しばく」ことによって消化する、新感覚のストレス解消エンターテインメントアプリです。
「スパジャム道場 2026 ～育成型ハッカソンで実践スキルを身につけよう～」で制作しました
https://www.craftstadium.com/hackathon/spajam-dojo-2026-1

## 主な機能と遊び方

### 1. ホーム画面 (Home)
アプリの入り口です。
- **ゲーム開始**: 「しばき始める！」ボタンで、ストレス発散ゲームを開始します。
- **時間設定**: ゲームの制限時間を **10秒・30秒・60秒** から選択できます。
- **カスタム写真設定**: 「写真を選んでしばく」から、スマホ内の写真（上司、嫌いな野菜、エラー画面など）をアップロードして、それをしばく対象に設定できます。

### 2. ゲーム画面 (Game)
ひたすら連打する画面です。
- **操作**: 画面中央のボタン（または設定した写真）をひたすらタップ（連打）します。
- **演出**:
    - **CHAOSモード**: タップするたびに画面が激しくフラッシュし、オノマトペ（バシッ！ドカッ！）が飛び散ります。
    - **バイブレーション**: タップに合わせてスマホが振動し、打撃感を演出します。
- **ゴール**: 制限時間内にどれだけ多く「しばける」か、ハイスコアを目指します。

### 3. 結果画面 (Result)
ゲーム終了後の画面です。
- **スコア表示**: あなたの連打回数（しばき回数）が表示されます。
- **ランク評価**: 回数に応じて「神レベル」「ストレス溜まってる？」などの評価が下されます。
- **ネクストアクション**:
    - 「もう一度しばく」：リトライします。
    - 「情報提供する」：ハプニングを共有します。

### 4. 情報提供・投稿 (Post)
あなたの身に起きたハプニングを共有します。
- **写真・動画**: 証拠となる写真や動画をアップロードできます。
- **内容**: 「何があったか」をテキストで入力します。
- **位置情報**: ワンタップで現在地を取得し、ハプニング現場を記録します。

### 5. タイムライン・マップ (Timeline)
みんなのハプニングを共有する場所です。
- **マップ表示**: 日本地図上にみんなのハプニングがピンで表示されます。
- **リスト表示**: タイムライン形式で投稿を閲覧できます。
- **共感（しばく）**: 他人の投稿にある「しばく」ボタンを押して、代わりにしばいてあげる（共感する）ことができます。
- **セキュリティ**: 投稿された画像には「スクショ禁止」の透かしが入り、保存や撮影を心理的に防止します。

## PWA (Progressive Web App) 化について

このアプリはPWAに対応しており、スマートフォンにインストールしてネイティブアプリのように使用することができます。

### インストール手順

#### iPhone (iOS / Safari)
1.  SafariでアプリのURLを開く。
2.  画面下部の「共有」ボタン（四角から矢印が出ているアイコン）をタップ。
3.  メニューをスクロールして「ホーム画面に追加」をタップ。
4.  右上の「追加」をタップ。

#### Android (Chrome)
1.  ChromeでアプリのURLを開く。
2.  画面右上のメニュー（︙）をタップ。
3.  「アプリをインストール」または「ホーム画面に追加」をタップ。
4.  指示に従って追加。

これでホーム画面にアイコンが追加され、全画面でアプリを楽しむことができます！
