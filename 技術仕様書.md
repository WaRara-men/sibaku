# Shibaku!! - ユーザーマニュアル・仕様書

## 1. アプリ概要

Shibaku!!はストレス発散を目的としたシンプルで直感的な娯楽アプリです。
日常のストレスやイライラを、画面をタップして「シバく」ことで解消できます。

## 2. 基本プレイ方法

### 2.1 ホーム画面

アプリを起動すると、以下の要素が表示されるホーム画面が表示されます：

**「Shibakuを開始」ボタン**

* 中央に配置された大きなボタン

* タップするとデフォルト設定（30秒）でゲーム開始

* ボタンは目立つ赤色でデザインされ、タップを促すアニメーション

**時間選択オプション**

* ゲーム開始前に選択可能な3つの時間帯：

  * 10秒：短時間でサクッとストレス発散

  * 30秒：標準的なプレイ時間（デフォルト）

  * 60秒：じっくりと長く楽しみたい方向け

* ラジオボタンまたはタブ形式で選択

**「シバく写真を選択」機能**

* カメラロールから写真を選択するボタン

* 選択可能な写真：上司の写真、嫌いなもの、ストレスの原因など

* 選択した写真がゲーム画面の中央ボタンに表示される

* 未選択時はデフォルトの「シバく」ボタンが表示

### 2.2 ゲーム画面

**基本操作**

* 画面中央のボタン（またはカスタム写真）を素早く連続タップ

* タップするたびに「シバいた！」のカウントが増える

* 残り時間が画面上部に表示（カウントダウン形式）

**ビジュアル効果**

* タップ時の画面フラッシュ：白い光が一瞬画面全体を覆う

* CHAOSモード：一定のタップ数に達すると画面が赤く光り、「CHAOS!!」の文字が表示

* 振動フィードバック：各タップで軽い振動（端末設定に依存）

* タップ音効果：「パン！」「バシッ！」などの効果音

**スコアシステム**

* 1タップ = 1ポイントの基本スコア

* コンボボーナス：連続タップでボーナス倍率が上昇

* CHAOSモード時は2倍のスコア獲得

### 2.3 結果画面

**スコア表示**

* 大きな文字で総タップ数を表示

* ランク評価：

  * 神レベル（1000回以上）：金色の冠マーク

  * 達人レベル（500-999回）：銀色の星マーク

  * 玄人レベル（200-499回）：銅色の円マーク

  * 凡人レベル（100-199回）：青色の丸マーク

  * 初心者レベル（0-99回）：灰色のハートマーク

**アクションボタン**

* 「もう一度挑戦」：同じ設定でもう一度ゲーム開始

* 「ホームに戻る」：ホーム画面に戻る

* 「情報を提供」：インシデント報告画面へ移動

## 3. インシデント報告機能

### 3.1 投稿・報告画面（「情報を提供」）

**写真/動画アップロード**

* カメラで撮影またはギャラリーから選択

* 複数枚の写真投稿可能（最大5枚）

* 動画は15秒以内に制限

* プレビュー機能で確認してから投稿

**テキスト説明**

* 起きた「ハプニング」を説明するテキスト入力欄

* 最小10文字、最大500文字

* ハッシュタグの自動補完機能

**位置情報**

* GPSを使用して自動的に位置情報を取得

* 地図上で正確な位置を確認・調整可能

* 位置情報の非公開オプションも提供

### 3.2 タイムライン・マップ画面

**地図表示モード**

* Google Maps上にピンで投稿を表示

* ピンの色で感情を表現：

  * 赤：怒り・イライラ

  * 青：悲しみ・落胆

  * 黄：驚き・呆れ

* ピンをタップすると投稿の概要表示

* 現在地から近い順に表示オプション

**リスト表示モード**

* 時系列順の投稿一覧

* 各投稿に表示される情報：

  * サムネイル画像

  * タイトル（最初の30文字）

  * 投稿日時

  * シバかれた回数

  * 位置情報（市区町村レベル）

**インタラクション**

* 各投稿に「シバく」ボタンを配置

* タップするとアニメーションと共にカウント増加

* 1日1投稿につき1回まで「シバく」可能

* シバいた回数が多い投稿は「ホット」マーク付与

### 3.3 セキュリティ対策

**スクリーンショット防止**

* スクリーンショット検知機能

* 検知時に警告メッセージ表示

* 投稿に透かし（ウォーターマーク）自動付与

* 連続スクリーンショット試行時は一時的に利用制限

**プライバシー保護**

* 顔認識で自動モザイク処理オプション

* 個人を特定できる情報の自動検出

* 投稿前の内容確認画面必須

## 4. 操作フロー

### 基本プレイフロー

1. アプリ起動 → ホーム画面表示
2. 時間選択（任意） → 「Shibakuを開始」タップ
3. ゲーム画面で素早くタップ → スコア獲得
4. 時間終了 → 結果画面でスコア確認
5. リトライまたはホームに戻る

### インシデント報告フロー

1. 結果画面またはホームから「情報を提供」選択
2. 写真/動画を選択または撮影
3. テキスト説明を入力
4. 位置情報を確認・調整
5. 内容を確認して投稿
6. タイムライン/マップで他の投稿を閲覧

## 5. 技術仕様

**対応デバイス**

* iOS 14.0以上

* Android 8.0以上

* タブレット対応（横向き・縦向き両対応）

**必要な権限**

* カメラ：写真撮影用

* 写真ライブラリ：画像選択用

* 位置情報：投稿時の位置情報取得用

* 通知：新機能やアップデート情報の配信用

**オフライン機能**

* ゲームプレイはオフラインで可能

* 投稿はオンライン時に自動同期

* ローカルに一時保存（最大10件）

## 6. 注意事項・免責事項

* 過度なタップは端末の破損原因となる可能性があります

* 公共の場での使用は周囲の配慮をお願いします

* 投稿内容は不適切と判断された場合、削除される可能性があります

* 位置情報は必ずしも正確ではない場合があります

* 本アプリはストレス発散を目的とし、暴力を助長するものでは
